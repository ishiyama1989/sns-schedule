# SNSチーム スケジュール管理（プロトタイプ）

SNS運用代行チーム向けの、共有スケジュール + メンバー空き状況 + 時給/支払い管理アプリです。

## 起動方法

```bash
cd sns-schedule
npm install      # 初回のみ
npm run dev      # http://localhost:5173 （または表示されたURL）
```

## お試しアカウント

| 役割 | メール | パスワード |
| --- | --- | --- |
| オーナー | owner@example.com | owner123 |
| メンバー | yamada@example.com | member123 |

新規登録すると「メンバー」として追加されます。

## 機能

- **📅 カレンダー**（全員）: いつ・どこで・誰が・何をするかを共有。日付をクリックして予定を追加・編集・削除。種別は「稼働 / 撮影 / 納品」。その日に空いているメンバーも表示。
- **🟢 自分の空き日**（全員）: 日付をクリックして自分の空き日を登録。
- **👥 メンバー・時給**（オーナーのみ）: メンバーごとの時給を設定。
- **💰 支払い集計**（オーナーのみ）: 3ヶ月（四半期）ごとに「稼働・撮影の時間 × 時給」を自動集計。

## 現在の制約（プロトタイプ）

- データはブラウザ内（localStorage）に保存。**端末をまたいだ共有や本物のログインはまだできません**。
- 本番では バックエンド + データベース（例: Supabase）に置き換えて、実際のチーム共有・認証を実装します。

## 技術構成

Vite + React + TypeScript。データ操作は `src/store.ts` に集約しているため、ここをサーバAPIに差し替えれば本番化できます。
